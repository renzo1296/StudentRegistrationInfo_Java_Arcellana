# StudentRegistrationInfo_Java_Arcellana

A simple Java-based Student Registration Information System developed for academic purposes. This project allows users to input, manage, and display student registration details using Java programming concepts such as classes, objects, methods, conditional statements, and user input handling.

## Features

* Add student information
* Display registered student details
* Simple and user-friendly console interface
* Demonstrates basic Object-Oriented Programming (OOP) concepts in Java

## Technologies Used

* Java
* IntelliJ IDEA / NetBeans / Eclipse
* Console-based application

## Purpose

This project was created as part of a programming exercise and learning activity for Java Object-Oriented Programming.
